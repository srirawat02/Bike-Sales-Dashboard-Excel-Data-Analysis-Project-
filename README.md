# Bike-Sales-Dashboard-Excel-Data-Analysis-Project-

## 📖 Project Overview

This project analyzes customer data related to bike purchases and presents insights through an interactive Excel dashboard.

The analysis focuses on understanding which types of customers are more likely to purchase bikes based on factors such as:

- Income
- Gender
- Age
- Commute Distance
- Marital Status
- Education
- Region

The dataset was cleaned, transformed, and analyzed using Excel Pivot Tables, Pivot Charts, and Interactive Slicers to create a dynamic dashboard that helps identify patterns in bike purchasing behavior.

## 🎯 Purpose of the Project

Businesses selling bikes need to understand:

- Who their most likely customers are
- Which demographic groups buy bikes the most
- How income and commute distance affect purchasing decisions
- Which age groups are more interested in buying bikes

This project helps demonstrate how Excel can be used to analyze customer behavior and support marketing decisions.
